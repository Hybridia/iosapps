Privacy Policy

Your Privacy Matters: MediPlanner is designed with privacy-first principles. All your health data stays on your device and is never transmitted to external servers.

Information We Collect

Health and Medical Information

MediPlanner stores the following information locally on your device:

  • Appointment Details: Date, time, type of appointment (doctor visit, dentist, specialist, etc.)
  • Healthcare Provider Information: Doctor names, clinic locations, phone numbers
  • Medical Notes: Personal notes and reminders you add to appointments
  • Medication Information: Medication names, dosages, and reminder schedules
  • Recurring Schedule Data: Frequency and pattern settings for recurring appointments
  • Notification Preferences: Reminder settings and notification timing preferences

Device Information

  • Local Storage: App preferences and settings stored in device UserDefaults
  • Theme Preferences: Color scheme and personalization settings (Pro version only)
  • Notification Access: Permission to send local notifications for appointment reminders

How We Use Your Information

Core Functionality (All Versions)

  • Display and manage your medical appointments
  • Send local notifications for appointment reminders
  • Generate recurring appointments based on your schedules
  • Organize appointments by type and date
  • Provide timeline and calendar views of your appointments

Pro Version Exclusive Features

  • Export appointments to .mediplanner files for backup or sharing
  • Import appointments from .mediplanner files shared by other users
  • Custom theme colors and advanced personalization options

Import and Export Functionality (Pro Version Only)

Pro Version Exclusive Feature

Data Sharing Through Files: Only the Pro version of MediPlanner includes import and export functionality. This allows Pro users to export their appointments to a .mediplanner file and import appointments from files shared by other Pro users. This functionality is entirely user-controlled and optional.

Export Features

  • Local File Creation: Export creates .mediplanner files stored in your device's temporary directory
  • JSON Format: Files contain appointment data in structured JSON format with metadata
  • Sharing Control: You choose how and with whom to share exported files
  • File Contents: Exported files include appointment details, recurring schedules, and metadata (export date, app version)
  • No Automatic Upload: Files are not automatically transmitted anywhere

Import Features

  • File Validation: App validates .mediplanner file format and version compatibility
  • Duplicate Detection: Identifies potential duplicate appointments before import
  • User Control: You decide which appointments to import and how to handle duplicates
  • Notification Options: Choose whether to import original notification settings or apply new defaults
  • Import Preview: Review all appointments before confirming import
  • Undo Capability: Can undo recent imports to remove imported data

File Security and Privacy

  • Temporary Storage: Import files are temporarily stored and cleaned up after processing
  • Local Processing: All import/export operations happen locally on your device
  • User Responsibility: You control who has access to exported files and their contents
  • No Cloud Integration: No automatic syncing or cloud storage of exported files

Data Storage and Security

Local Storage Only

All your data remains on your device. MediPlanner uses iOS UserDefaults and local JSON encoding to store your information. We do not transmit, upload, or sync your health data to any external servers or cloud services.

Data Security Measures

  • Device-Level Security: Your data is protected by your device's built-in security features
  • No Network Transmission: Health data never leaves your device
  • Local Encryption: Data is encoded using standard iOS security practices
  • App Sandbox: Information is isolated within the app's secure container

Notifications and Permissions

Local Notifications

MediPlanner requests permission to send local notifications to remind you of upcoming appointments. These notifications:

  • Are generated locally on your device
  • Do not transmit data to external services
  • Can be customized or disabled in app settings
  • Include appointment titles and reminder times you've set

Data Sharing

No Automatic Data Sharing: MediPlanner does not automatically share your data with third parties, advertising networks, or analytics services. All data sharing is explicitly controlled by you through the import/export features.

User-Controlled Sharing (Pro Version Only)

  • Export Files: Pro users can voluntarily export and share .mediplanner files containing their appointment data
  • Sharing Responsibility: When Pro users share exported files, they are responsible for ensuring recipient trustworthiness
  • File Contents Warning: Exported files contain all appointment details including sensitive health information
  • No Tracking: We do not track or monitor file sharing activities
  • Free Version: The free version does not include any data sharing capabilities

Data Retention and Deletion

User Control

  • Full Control: You have complete control over your data
  • Individual Deletion: Delete specific appointments or recurring series
  • Complete Removal: Use the "Clear All Data" feature in settings to remove all information
  • Import Management (Pro Only): Undo recent imports to remove imported appointments
  • File Cleanup (Pro Only): Temporary import files are automatically cleaned up after processing
  • App Deletion: Uninstalling the app removes all stored data from your device

Version Differences

Free Version

  • Core appointment management features
  • Basic notification system
  • Default theme only
  • Local data storage only
  • No import/export capabilities
  • No data sharing functionality

Pro Version

  • All Free version features
  • Import/Export functionality - Create and share .mediplanner files
  • Custom theme colors - Personalize app appearance with multiple color schemes
  • File-based data sharing capabilities
  • Same local-only data storage policy as Free version
  • Additional privacy controls for import/export operations

Children's Privacy

MediPlanner is designed for users of all ages. We do not knowingly collect personal information from children under 13. If the app is used by minors, we recommend parental supervision. All privacy protections apply equally regardless of user age.

Changes to This Policy

We may update this Privacy Policy periodically to reflect changes in our practices or for legal compliance. We will notify users of significant changes through app updates or in-app notifications.

Your Rights

Since all data is stored locally on your device, you have complete control over your information:

  • Access: View all your stored appointments and settings within the app
  • Modification: Edit or update any information at any time
  • Deletion: Remove individual items or all data through app settings
  • Export (Pro Only): Create backup files of your appointment data
  • Import Control (Pro Only): Choose which appointments to import from shared files
  • Sharing Control (Pro Only): Decide when and with whom to share your exported data
  • Calendar integration (Pro Only): Plannings you make in MediPlanner can also be integrated with Calendar app.
